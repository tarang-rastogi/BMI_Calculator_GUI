# 🧮 BMI Calculator GUI (Python)

## 📌 Description
1. This project is a GUI-based BMI (Body Mass Index) Calculator developed using Python and Tkinter.  
2. The application allows users to enter their name, weight, and height to calculate their BMI and determine their health category.
3. The system also stores user data in a CSV file, enabling users to view previous records and perform trend analysis using graphical visualization.
4. This project demonstrates the use of GUI development, file handling, and data visualization in Python.

## 🚀 Features
- User-friendly GUI
- Calculate BMI instantly
- Store user data (CSV file)
- View history
- Graph visualization using matplotlib

## 🛠️ Technologies Used
- Python
- Tkinter
- Matplotlib
- CSV

## ▶️ How to Run
1. Install Python
2. Install required libraries:
   pip install matplotlib
3. Run the file:
   BMI_App_through_GUI.py
