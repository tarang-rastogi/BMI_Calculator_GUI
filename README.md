# 🧮 BMI Calculator GUI (Python)

## 📌 Description
This is a GUI-based BMI (Body Mass Index) Calculator built using Python and Tkinter.  
It allows users to calculate BMI, store data, and visualize BMI trends.

## 🚀 Features
- User-friendly GUI
- Calculate BMI instantly
- Store user data (CSV file)
- View history
- Graph visualization using matplotlib

## 🛠️ Technologies Used
- Python
- Tkinter
- Matplotlib
- CSV

## ▶️ How to Run
1. Install Python
2. Install required libraries:
   pip install matplotlib
3. Run the file:
   python main.py
